# HTML & CSS Project

This project is a simple web application that demonstrates the use of HTML and CSS to create a styled web page.

## Project Structure

```
html-css-project
├── src
│   ├── index.html
│   └── styles
│       └── style.css
└── README.md
```

## Files Description

- **src/index.html**: Contains the HTML structure of the web page and links to the CSS file for styling.
- **src/styles/style.css**: Contains the CSS styles for the HTML elements defined in `index.html`.

## Getting Started

To set up and run this project, follow these steps:

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in a web browser to view the web page.

## Customization

Feel free to modify the HTML and CSS files to customize the web page according to your needs.
